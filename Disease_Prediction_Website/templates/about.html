About Disease Prediction

This project is based on the predictive ability of Machine Learning. It is capable of predicing a disease from its symptoms. A user is required to populate the symptoms he/she is experiencing from the provided select boxes. It takes about 10 to 20 secomds to produce the most possible disease. After the predctions, a user may connect to the nearest doctor. The symptoms stores every prediction outcome for future reference.

Phases of development
Model Developement
Support Vector Classifier model of the sklearn.svm package is developed.
Training
The model is trained on a dataset containing various diseases and their sysmptoms.
Testing
The functionality of the model was tested using the real disease dataset. Average Performance was about 96.99%.
Deployment
The system is deployed as a website using Python`s Flask framework.


The functionality of this model is based on the accuracy of the dataset, to achieve better formance accurate data is needed.
